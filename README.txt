/************************   Use Your Old Pom.xml File ************************************************?



/*****************************************************************************************************/

/*****************************************************************************************************/
2. copy json templates inside ./src/main/resources/jsonTemlate/ folder

/*****************************************************************************************************/

3. run 'JsonToExcel.java' to generate excel files for json data

/*****************************************************************************************************/

4.  the excel results will be placed in './src/main/resources/results/' folder

/*****************************************************************************************************/

5.  edit the excel values.

/*****************************************************************************************************/

6. run 'ExcelToJson' to generate list of json files for each row of each excel based on the template data.
